# 词云中文字体（可选）

将 **Noto Sans SC Regular** 放入此目录后，词云会优先使用，避免依赖网络下载。

- 下载地址：[Google Fonts - Noto Sans SC](https://fonts.google.com/noto/specimen/Noto+Sans+SC) → 下载后解压，将 `NotoSansSC-Regular.otf` 放入本目录。
- 或直接下载：<https://github.com/google/fonts/raw/main/ofl/notosanssc/NotoSansSC-Regular.otf> 并保存为 `NotoSansSC-Regular.otf`。

未放置字体时，应用会尝试从系统或网络获取中文字体。
