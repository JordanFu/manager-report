# 让任何人通过网页访问本应用（约 2 分钟）

部署到 **Streamlit Community Cloud** 后，你会得到一个公网链接（如 `https://xxx.streamlit.app`），分享给任何人即可使用，无需安装。

---

## 操作步骤

### 1. 打开部署页面

在浏览器中打开：  
**[https://share.streamlit.io](https://share.streamlit.io)**

### 2. 用 GitHub 登录

- 点击右上角 **Sign in**，选择 **Continue with GitHub**
- 使用你的 GitHub 账号（JordanFu）授权

### 3. 新建应用并填写本仓库

- 点击右上角 **“New app”** 或 **“Create app”**
- 若提示 “Do you already have an app?”，选 **“Yup, I have an app”**
- 在 **Paste GitHub URL** 中粘贴本仓库地址：

```
https://github.com/JordanFu/manager-report
```

- **Main file path** 填：`app.py`（若界面有该项）
- 点击 **“Deploy”**

### 4. 等待构建

通常 1～3 分钟。完成后会显示你的应用链接，形如：

**https://xxxxx.streamlit.app**

把该链接发给任何人，对方用浏览器打开即可上传问卷底表、查看报告。

---

## 若链接失效

- 直接访问：https://share.streamlit.io ，登录后可在 “My apps” 中查看或重新部署。
- 本仓库已包含 `requirements.txt` 与 `app.py`，无需再改代码即可部署。
