# 管理者调研报告

基于问卷底表（.xlsx / .csv）生成**管理者调研报告**的 Streamlit 应用，支持全局概览、维度深度分析、个人详细报告、开放反馈与异常提醒。

## 功能概览

- **数据上传**：左侧边栏上传问卷底表（五级量表，含「填写人」/「姓名」列）
- **全局概览**：核心数据、维度均分、词云与分布
- **维度深度分析**：各维度得分与行为项明细
- **个人详细报告**：选定学员的雷达图、维度得分与开放题
- **开放反馈**：开放题汇总与词云
- **异常提醒**：统一分值等异常记录

## 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

浏览器打开提示的地址（一般为 http://localhost:8501），在左侧上传问卷底表即可。

## 部署为公开页面

将本仓库部署到 [Streamlit Community Cloud](https://share.streamlit.io) 即可获得公网链接，任何人通过浏览器上传自己的底表即可使用。详细步骤见 **[部署说明.md](部署说明.md)**。

## 项目结构

| 文件 | 说明 |
|------|------|
| `app.py` | Streamlit 主应用 |
| `config.py` | 维度、题目映射与分值配置 |
| `data_processor.py` | 数据清洗与得分计算 |
| `requirements.txt` | Python 依赖 |
| `.streamlit/config.toml` | Streamlit 主题与配置 |
| `wordcloud_mask.png` | 词云蒙版图（可选） |

## 配置说明

- 问卷维度、题目与行为项的映射在 **`config.py`** 中，可按实际题本修改。
- 表头需包含「填写人」或「姓名」列，选项为五级量表（如：总是如此 / 经常如此 / 有时如此 / 很少如此 / 从未展现）。

## 许可证

按项目需要自行选择。数据与报告内容由使用方自行负责。
